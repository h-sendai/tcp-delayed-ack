# TCP Delayed Ack

- サーバーは起動して、read()で待つ。
- クライアントは起動して接続するとwrite()で1バイト書き、
  続けてread()で待つ。
- サーバーは1バイトread()して1バイトwrite()する。
- クライアントはread()で1バイト読む。
- クライアントは1秒sleep()する。
- 以下繰り返し。何回繰り返すかは-nオプションで指定する。

```
クライアント            サーバー

1バイト ----------->
        <----------- 1バイト

(スリープ1秒)

1バイト ----------->
        <----------- 1バイト
```
